Comment héberger un site Wordpress sur HEROKU

INTRODUCTION

Salutation ! Aujourd'hui nous allons voir comme vu dans le titre comment mettre un site Wordpress sur HEROKU. Un petite intruction ne fera pas de mal. HEROKU est une plateforme de deploiement d'application web sur un Cloud, il fonctionne comme un hébergeur web qui propose d'héberger des sites web.

INSTALLATION

1- Inscription sur HEROKU

Aller sur le site de HEROKU et cliquer sur Sign up

Un mail de confirmation à valider vous sera envoyé.

Votre inscription sur le site est terminé

2- Créer un repo Github

  1. Aller sur github
  2. S'inscrire (ou se connecter si vous avez déjà un compte)
  3. Aller sur le + en haut à droite et cliquer sur "new repository"
  4. Choisir un nom pour le repo
  5. Tout le reste est au choix (publique ou privée, ajouter un READ.ME ou non)
  6. Cliquer sur "create a repository"

Votre repo est créer HOURRA!

3- Mettre le site WordPress sur Github

a- installer git sur votre ordinateur

  1. Aller sur le site de git
  2. Cliquer sur le Downloads rouge
  3. choisir ensuite la version de git pour votre system d'exploitation (Windows, Linux ou Mac et 64bit/32bit)
  4. sous windows faire un double clique sur le fichier télécharger
  5. Sous Linux (debian/Ubuntu) dans le terminaln entrer la ligne de commde suivante #sudo apt-get install git
  6. si git est déjà installé sur votre ordinateur, vous dans le terminal utiliser la ligne de commande suivante: #git clone https://github.com/git/git

b-envoyer votre wordpress sur github

  1. Ouvrir le terminal
  2. ce placer dans le dossier de votre site wordpress $ cd non du dossier wordpress
  3. initialiser le git avec la ligne de commande $git init
  4. créer un remote: $ git remote add origin https://github.com/username/non de votre dossier WP
  5. et ensuite pousser votre dossier sur github $ git push origin master

3- Heberger votre Wordpres sur heroku

  1. Aller sur heroku
  2. Aller sur "NEW" en haut à droite et cliquer sur "Create New application"
  3. Entrer un nom pour l'application et la région
  4. Aller sur l'onglet deploy
  5. à la section "deployment method" et selectionner à son github
  6. cliquer ensuite sur "search" et selectionner votre repo WordPress
  7. il faudra ensuite déployer la brache
  8. à la section Manual deploy
