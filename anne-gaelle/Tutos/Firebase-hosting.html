Firebase Hosting

Tutoriel noob 101

Création du projet

  1. Aller sur le panneau de contrôle de Firebase . Se connecter si cela n'est pas encore fait.
  2. Cliquer sur le afin de créer et nommer un nouveau projet.

Installer Firebase CLI

A présent il faut utiliser Firebase en ligne de commande. Il faut avoir Node.js et npm d'installé sur la machine.

  1. Installer Node.js. La version 6 minimum est obligatoire.
  2. Installer Firebase CLI via NPM avec la commande suivante : 
     npm install -g firebase-tools
  3. Se connecter à Firebase avec son compte Google en entrant la commande : 
     firebase login
  4. Vérifier si la connection est fonctionnelle et si l'accès aux projets est possible : 
     firebase list
    Cette commande liste les différents projets disponibles dans le panneau de contrôle.
  5. Mettre à jour la version de CLI en entrant à nouveau la commande : 
     npm install -g firebase-tools

Initialiser le projet

  1. Pour connecter le projet local au projet Firebase, lancer la commande suivante depuis la racine du projet sur votre machine : 
     firebase init
  2. Dans la console, choisir l'option concernant le hosting. La touche espace sert à se déplacer et entrée sert à valider le choix.
  3. Sélectionner le projet Firebase à relier au projet local.
  4. [Optionnel] Spécifier un répertoire publique pour l'emplacement de la racine Firebase. Ce dossier contiendra tous les fichiers servis ainsi que le index.html qui seront déployés sur l'hébergeur.
    Par défaut l'initialisation crée un fichier public, cela peut être modifié dans le fichier firebase.json.
  5. Choisir la configuration du site à déployer : "one-page app".

Déployer le projet

  1. Pour déployer le site, il faut entrer la commande suivante à la racine du projet local : 
     firebase deploy
  2. Cette commande déploie une version du site à l'adresse de Firebase par défaut soit 
    id-projet-firebase.firebaseapp.com
  3. Pour être sûr du push des bons fichiers, déplacer le index.html et autres fichiers dans le dossier public et relancer la commande deploy.

And voilà !

Le site est déployé et le tuto est terminé !
